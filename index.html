JS Console

A dependency free JavaScript Console that can be embeded in any site.

Getting Started

Import the script and style into the head of your document.
<script src="jsconsole.js"></script>
<link rel="stylesheet" href="jsconsole.css" />
Add this HTML to your page:
<div class="jsconsole"></div>
This is automatically created:
Keyboard Controls
Enter
Runs the code, unless using the data-run-btn attribute (see below).
Shift + Enter
Creates a new line.
Up
If your on the top line of your code, pressing the up key moves backworks through your code history.
Down
If your on the bottom line of your code, pressing the down key moves forewards through your code history.

Default Code

To pre-populate the console with some code, put it within the <div>:

<div class="jsconsole">var foo = "bar"</div>
var foo = "bar"

You can use a <pre> element to preformat your code (line breaks and spacing preserved).

<div class="jsconsole"><pre>(n => {
  return n + 2;
})(2)</pre></div>
(n => {
  return n + 2;
})(2)
Preserve the Default Code

Give the <div> element a data-preserve-default-code attribute to automatically reload the default code after each time code is ran.

<div class="jsconsole" data-preserve-default-code>true == !1</div>
true == !1

Run button

To show an "Run" button give your <div> element a data-run-btn attribute. Doing this will also prevent the enter key from running the code.

<div class="jsconsole" data-run-btn></div>
1 + 2

Give the data-run-btn attribute a value to set a custom "run" button text.

<div class="jsconsole" data-run-btn="Execute"></div>
2 + 2

Auto Run Default

You can have the console automatically run teh default code by giving it an attribute of data-run-default.

1 + 7

Interactive Outputs

Objects, Arrays and HTMLElements have interactive ouputs that allow you to expand/collapse data to optimise space while making data available.

Arrays

The first three items of an Array are visible, then the array can be expanded to show all items. The size of the array is displayed muted before the array.

<div class="jsconsole" data-run-btn data-preserve-default-code>["item 1", undefined, null, 2, true, "item 6"]</div>
["item 1", undefined, null, 2, true, "item 6"]
Objects

The first three properties of an Object are visible, then the object can be expanded to show all properties.

{hello:"world",foo:"bar",key3:"value 3",key4:"value 4","key 5":"value 5"}
HTML Elements

HTMLElements are displayed in their XML format, block elements with block children are collapsed but can be expanded to show the contents.

<div id="myDiv" style="display: none">
  <p>Hello World</p>
</div>
<div class="jsconsole" data-run-btn data-preserve-default-code>document.getElementById("myDiv")</div>

Hello World

document.getElementById("myDiv")

Syntax Highlighting

To use syntax highlihgting just include Highlight JS on your page. JS Console will automatically apply the VS2015 theme.

<script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/9.15.10/highlight.min.js"></script>
